Number Guesser Pt. 1

Learn basic Python concepts by building a program that selects a random number that you have to guess.

Hi everyone, today I want to go over a small beginner Python project that will teach you many important language concepts. In this project we will build a program that picks a random number that you have to guess. The program will teach novice programmers about functions, passing variables, the Random module, and much more.

To start, let’s outline what this program will look like. I outlined my program on a digital white board for organization purposes. Basically, this drawing outlines four steps. One: the program will load. Two: the computer will pick a random number in a pre-defined range. Three: the user will guess the number until they get it right. Finally, once the user gets the number right, the program will notify the user and close itself.

Now that we have a worded and visual idea of the structure of this code, let’s actually program it!

import random

def randomize():

def guess_it():

if __name__=="__main__":
    randomize()
    guess_it()
    print("Congrats! You won")

So, what’s going on here? I’ve created three functions: ‘If __name__==“__main__”’, ‘guess_it’, and ‘randomize’. The first of the aforementioned functions basically tells the computer to run the code if this is the primary file run, i.e. functions aren’t being called by other files. We will use the ‘randomize’ function to determine the random number the user has to guess. Finally, the ‘guess_it’ function is where the user will make their guesses. At the top of the file I’ve imported the Random module. This module is pre-installed during your Python download, it will allow us to generate random numbers.

Now that we have the code outline lets get coding. First let’s code the randomize function. To start let’s define a variable to take the value of the randomly generated number. We’ll set that equal to a function in Random called ‘randint’, this function will generate a random integer in a range that we’ll specify as 1 to 100. Finally let’s return X and save it to the variable set to the function value. This is so the value can be accessed by other functions through the program.

def randomize():
    x = random.randint(1,100)
    return x

if __name__=="__main__":
    num = randomize()
    guess_it(num)

Now we can start coding the user end of this program. It is essential that we realize that the program does not move to the next line in the main function until we exit this function. That means once this function is completed, we know the program has completed. Because of that we don’t need to return any values, instead if we exit a for loop when the user guesses the right number, we can exit the function.

First let’s define a variable, we’ll call it guessed, it's a boolean that represents if the user has guessed the right number. Obviously the initial value should be ‘False’ as the user has not guessed the correct number. Next let’s enter a while loop under the condition that guessed remains false. To get user input, we’ll use the input function. So far your code should look like this:

def guess_it(num):
    guessed = False
    while not guessed:
        guess = input("guess the number: ")

Now comes the difficult part, what do we do with the code? Well, there are four options for user input.

  1. The guessed value is higher than the computer’s number
  2. The guessed value is lower than the computer’s number
  3. The guessed value is equal to the computer’s number
  4. The guessed value is not a number

Because the input function accepts input as a string, it can’t be compared to an integer value. To convert a string value into an integer, we can use the int() function. Unfortunately, the int() function does not work for all string values: if you try to use it on a letter, it will return an error. Therefore, we must use a try/except structure to test our values. First, we will enter the try area of our code.

In the try area of our code we can attempt if statement. This if statement will test if the inputted value is lower, higher, or equal to the computer’s value. In each instance, we will hint the user in the direction of the actual number if they are incorrect. If the if statement’s conditions are not met, we enter the except area of our code. Once the user guesses the correct number we will change the value of guessed to True. This will exit the loop and cause the program’s last line to end.

If we enter the except area of the code that only leaves the fourth option: the user did not not enter an integer value. Because of this, we will notify the user of their error and circle back to the top of our loop.

Here’s what the code should look like:

def guess_it(num):
    guessed = False
    while not guessed:
        guess = input("guess the number: ")
        try:
            if int(guess) > num:
                print("the number is lower")
            elif int(guess) < num:
                print("the number is higher")
            elif int(guess) == num:
                guessed = True
        except:
            print("you did not input a number")
